Introduction

This HTML document demonstrates the conversion capabilities of the MarkItDown API. The server can convert this HTML content into clean Markdown format while preserving the document structure.

Features

  • Headers - Converted to markdown headers
  • Paragraphs - Maintained as text blocks
  • Lists - Both ordered and unordered
  • Links - External links are preserved
  • Emphasis - Italic and bold text

Code Examples

The API can also handle code blocks:


def hello_world():
    print("Hello from MarkItDown!")
    return "Success"
